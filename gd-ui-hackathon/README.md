This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

1. `$ git clone https://github.com/gooddata/ui-sdk-examples.git`
1. `$ cd ui-sdk-examples/gd-ui-hackaton/`
1. `$ yarn install`
1. `$ yarn start`
